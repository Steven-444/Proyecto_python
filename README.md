Este proyecto consiste en un sistema de control de inventario desarrollado en Python. El programa utiliza una matriz para almacenar la información de diferentes artículos, incluyendo código, nombre, stock actual y stock mínimo requerido.

El sistema permite identificar cuáles productos necesitan ser reabastecidos mediante una función que calcula automáticamente la cantidad exacta a pedir según las reglas del inventario. Además, muestra información detallada de cada artículo, como la cantidad actual en inventario, cantidad mínima y máxima de abastecimiento y la cantidad necesaria para reponer el stock.

Este proyecto fue realizado como práctica de fundamentos de programación, aplicando conceptos como matrices, funciones, ciclos, condicionales y lógica de negocio en Python.
